## PROJECT DESCRIPTION

This project is a Java-based recommendation system that suggests
products or content based on user preferences. It uses Apache
Mahout library for collaborative filtering and provides recommendations
by analyzing user ratings on items.


## FEATURES

- User-based collaborative filtering
- Predicts ratings and suggests top items
- Uses Apache Mahout library
- Works with sample CSV data
- Console-based output


## TECHNOLOGIES USED

- Java
- Apache Mahout
- CSV for sample data
- Collaborative filtering


## FILE DETAILS

Program Name : RecommendationSystem.java  
Sample Data : data.csv (userID,itemID,rating)


## WORKING

The program reads user-item ratings from a CSV file, computes
similarities between users, and recommends items with the highest
predicted rating for a given user. It demonstrates the basics of
AI-based recommendation engines.


## OUTPUT

Displays recommended item IDs along with predicted ratings for a
specific user.


## CONCLUSION

This project demonstrates the implementation of a simple
recommendation engine in Java using Apache Mahout to provide
personalized suggestions based on user behavior.
